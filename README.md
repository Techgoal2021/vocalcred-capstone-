# 🚀 VocalCred: Zero-Barrier Reputation Protocol

**VocalCred** is a Reputation-as-a-Service platform designed for the informal economy. It allows artisans to build verifiable profiles using zero-barrier technologies like USSD and Voice, powered by AI.

## 📌 Problem
The informal artisan economy in Nigeria relies heavily on word-of-mouth. Artisans lack verifiable reputation scores, making it difficult for homeowners to trust them and for artisans to access formal services.

## ✨ Solution
VocalCred uses **Africa’s Talking USSD and Voice APIs** alongside **Google Gemini AI** to enable artisans to register and be discovered without needing a smartphone or data.

## 🛠️ Tech Stack
- **AI**: Google Gemini 1.5 Flash (Voice intent & extraction)
- **Communications**: Africa's Talking (USSD, Voice, SMS)
- **Frontend/API**: Next.js, Framer Motion, Tailwind CSS
- **Database**: Prisma + SQLite
- **Deployment**: Google Cloud Run (Dockerized)

## 🌟 Key Features
- **Zero-Barrier Intake**: Artisan registration via voice callbacks.
- **AI-Powered Search**: Natural language voice commands to find experts.
- **SMS Discovery**: Real-time delivery of verified artisan details via SMS.
- **Reputation Layer**: Verifiable trust scores for the informal sector.
